#搭建流程

Hexo 介绍

Hexo是基于NodeJs的静态博客框架，简单、轻量，其生成的静态网页可以托管在Github上。

  • 超快速度
  • 支持Markdown
  • 可以通过命令行一键部署

配置

网站的设置大部分都在_config.yml文件中，详细配置可以查看官方文档
下面只列出简单常用配置

  • title -> 网站标题
  • subtitle -> 网站副标题
  • description -> 网站描述
  • author -> 名字
  • language -> 网站使用的语言

添加评论

使用Gitment搭建评论系统

Gitment是一款基于Github Issues的评论系统。支持在前端直接引入，所以可以直接插入到markdown的文件末尾。可以在页面进行登录、查看、评论、点赞等操作，同时有完整的 Markdown / GFM 和代码高亮支持。
Gitment只支持通过github账号登陆。
基础使用

注册OAuth Application

获得一个新的OAuth Application。其他内容可以随意填写，但是注意要将callback URL填为评论页面对应的域名，如 https://rjx-965.github.io/
此时将会得到一个client ID 和一个 client secret，将被用于之后的用户登录。
